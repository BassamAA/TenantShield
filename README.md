# TenantShield

Full-stack web application built around tenant-related workflows, documentation, and protection tooling.

## Overview
TenantShield is a product-style web application designed to support tenant-facing workflows with a focus on clear user flows, document handling, and practical web application delivery.

## What it demonstrates
- Full-stack application structure using Next.js
- Product-oriented workflow design
- Integration of document and user-facing web features
- Practical experience building modern production-style web applications

## Tech Stack
- Next.js
- TypeScript
- React
- Stripe
- Resend

## Purpose
This repository is part of a broader portfolio of applications focused on building useful, production-style products with strong engineering foundations.
